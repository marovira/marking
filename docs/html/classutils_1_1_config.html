Detailed Description

A place-holder for all the configuration options of the main marking script.

Attributes

root: The root directory of the assignments. makeCSV: Whether to write the resulting marks to the connex formatted CSV file. makeComments: Whether to write the comment files for each student. workingDir: The working directory for the script. language: The language in which the assignments are written (currently one of Java or Python).
